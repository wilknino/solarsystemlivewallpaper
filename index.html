✦ Real-time shader shadow
Eclipse System

Solar & Lunar Eclipses
That Actually Look Real

The eclipse shadow is computed entirely in the fragment shader using geometric ray-projection — producing a physically accurate circular umbra spot on Earth's surface.

  • Solar eclipse casts a small circular dark spot on Earth — not a half-sphere, not a wedge. A real spot, centred on the sub-Moon point.
  • Lunar eclipse dims the Moon with a blood-red tint deep in the umbra — simulating Earth's atmosphere refracting orange-red light.
  • Both eclipse types detected using real astronomical alignment calculations from actual planet and moon positions.
  • Annotation badges — "Solar Eclipse ☀️" and "Lunar Eclipse 🌕" — appear with shadow cone labels when an eclipse occurs.
  • Eclipse geometry updates every frame in real-time as the simulation advances.
